Lab 3: Build a Website

Intro ITWS

Create a personal Intro to IT website in your RCS iit directory

  • The web site will be placed in an "iit" subdirectory of your RCS file space, and it must be RCS password protected: you will give rights to Varad (pathav@rpi.edu) and me (plotkr2@rpi.edu). Follow these instructions to set up your protected directory.
  • Guidelines:
    • You must have (at least) two pages: home and projects. Name these files:
      • index.html (this will be your landing page, or home)
      • projects.html
    • You must have a header, a menu, a content area, and a footer.
    • The only file in your root directory (at the root of your site) is the index.html page
    • All resources (e.g. CSS files) should be kept in a separate area(s) as per your design.
    • Populate your projects page with links to the Lab homework, beginning with your resume; continue to do this as we move forward.
    • As before, your code must be valid and properly indented. (And don't forget to set the <title> tag on each page.)
  • We will work through some of this together in class.
  • You will graded using the Lab 3 rubric.
  • As in Lab 2, here is an XHTML template you may start with. (And here is an HTML5 template you may also use.)
  • Feel free to experiment and be creative - and use the web as a reference!! (e.g. W3Schools)

  • This lab is due by the date on the LMS assignments page It must be posted to your RCS iit directory.

    Please create a plain text file, named README.md (or txt) file in your root directory and with comments about the lab, justifications for your chosen methods/styles/tech, etc. In the file, include the correct URL for your site.

    Include your outlines that you created in Monday's class (either reference as attachments or cut and paste)

    Create a zip file of your iit directory, name it - yourRCSid-ITWS1100-F21-Lab3.zip and submit it to the Assignments area of LMS. (in it will contain your site's root directory, your HTML, CSS, README, and any other files/resources relevant to your site)

    Include a working URL as a hyperlink to your RCS iit directory in the assignment comments and in your README files.

    The URL should look like:
    https://afsws.rpi.edu/AFS/home/yourNum/yourRcsId/public_html/iit/
